## Definición del problema 
El proyecto aborda un problema de clasificación de imágenes supervisado, cuyo objetivo es reconocer automáticamente el valor de un billete chileno a partir de una fotografía. Para ello se utiliza la librería ml5.js con el modelo MobileNet como extractor de características, aprovechando el aprendizaje previo de una red neuronal entrenada con imágenes. El sistema fue entrenado con un conjunto de imágenes organizadas en dos clases: billete de $5000 y billete de $10000, aprendiendo a diferenciar sus patrones visuales. Una vez entrenado, el modelo puede recibir una nueva imagen y predecir a qué clase pertenece, mostrando el valor del billete junto con el porcentaje de confianza.
